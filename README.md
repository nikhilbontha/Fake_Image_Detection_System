# Fake_Image_Detection_System
Fake Image Detection System that analyzes images using deep learning and forensic indicators such as texture patterns, edge density, lighting variation, and facial symmetry to detect AI-generated images.

This project is an AI-powered Fake Image Detection System built using Python, TensorFlow, and Streamlit. The system combines deep learning predictions with image forensic analysis techniques to identify whether an image is real or AI-generated.
The application analyzes several visual indicators including:
Texture variation (skin pores, fabric patterns, natural surfaces)
Edge density and compression artifacts
Lighting consistency and contrast distribution
Facial symmetry patterns
The system provides an interactive dashboard where users can upload or capture an image and receive:
AI probability prediction
Real vs Fake classification
Image forensic analysis
Confidence score
Detailed forensic report
This hybrid approach improves reliability compared to using deep learning alone.
